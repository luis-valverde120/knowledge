# Idea

Const is like a variables immutable in [[rust]] both can't allowed a value change, but have a some diferences.

- In **const** we can't allowed to use `mut`. This aren't just immutable by default this always is immutability.
- this can declare using the word `const` and his type of value need declare 

``` Rust
const PORT: i32 = 8000;  
```

## Concept

This are a value can't allowed change. this is generally defined in **uppercase**
This are valid in all time of program
